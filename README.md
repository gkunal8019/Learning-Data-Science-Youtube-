# 🚀 Generative AI Learning Roadmap

This repository contains curated resources to learn **Data Science, Deep Learning, and Generative AI** using structured YouTube playlists.

These resources focus on building strong fundamentals first and then moving toward advanced **GenAI and LLM applications**.

---

# 📚 Learning Path

Follow the learning sequence below for best results:

1. **Data Science Fundamentals**
2. **Deep Learning**
3. **Generative AI**

This order helps build strong foundations before working with advanced AI models. Deep learning itself is a subset of machine learning that uses layered neural networks to learn complex patterns from data. :contentReference[oaicite:0]{index=0}
